# **App Name**: SmartKharcha AI

## Core Features:

- User Authentication: Secure Google Sign-in to manage personal expense data.
- Intelligent Expense Capture: Add expenses instantly via text input, voice command, or by uploading a bill photo. An AI-powered tool will automatically extract the amount and assign a category from predefined options (Food, Transport, Bills, Shopping, EMI).
- Expense Listing & Management: View and manage all recorded expenses in a clear, sortable list.
- Monthly Budget Tracking: Set a monthly budget and track spending against it to stay on financial goals.
- Intuitive Dashboard: Access a simple dashboard displaying total spending and a graphical chart for quick financial overview.
- Data Persistence with Firestore: Securely save all user expense data and settings in Google Firestore.
- Multilingual Support & Dark Mode: Toggle between Hindi and English language options, and switch between a clean light and dark user interface mode.

## Style Guidelines:

- A dark color scheme provides a modern, sophisticated backdrop for financial management. The primary color, a deep blue-violet (#5252E0), represents stability and intelligence, ideal for a financial tracking app. This hue is intentionally vibrant against the dark canvas to draw attention to key data and actions. The background color, a very dark, desaturated blue-violet (#1F1F2E), ensures clarity and reduces eye strain. The accent color, a bright cool blue (#8CBFEE), creates clear contrast for interactive elements and highlights, reinforcing a clean, tech-forward aesthetic.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly techy, and precise feel, reflecting the AI aspect. Body font: 'Inter' (sans-serif) for excellent readability across different screen sizes and clear presentation of financial data.
- Utilize simple, clean line icons for clear and quick recognition of categories, navigation, and actions, suitable for a mobile-first experience.
- Implement a card-based layout for expenses and dashboard insights, prioritizing easy scanning and mobile-friendliness. Ensure clear visual hierarchy with ample negative space to maintain a 'clean UI' as requested.
- Incorporate subtle, smooth transitions for navigation between screens and loading states. Use micro-animations to provide feedback for user interactions like adding an expense or confirming a budget update.
