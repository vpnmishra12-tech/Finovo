
# SmartKharcha AI - User Guide 📱

SmartKharcha AI ek **Progressive Web App (PWA)** hai. Aap ise bina Play Store ke apne phone par install kar sakte hain.

## 🚀 How to Login (New System):
Humne login system ko asaan aur secure banane ke liye **Email & Password** setup kiya hai.

### 📝 Pehli Baar:
1. "Sign Up" tab par click karein.
2. Apna Email (ya koi bhi username format `test@test.com`) aur ek Password banayein.
3. Dashboard khul jayega.

### 🔐 Dubara Login:
1. "Login" tab par jayein.
2. Apna banaya hua Email aur Password daalein.

---

## 🛠 Important Setup (Admin):
Agar login fail ho raha hai, toh ye check karein:
1. [Firebase Console](https://console.firebase.google.com/) par jayein.
2. **Authentication** -> **Sign-in method** par click karein.
3. **"Email/Password"** provider ko **Enable** karein aur Save karein.

## 📲 Install on Phone:
1. Browser mein app kholien.
2. Menu (3 dots) par jayein aur **"Add to Home Screen"** select karein.
