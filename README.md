# SmartKharcha AI - User Guide 📱

Agar aapko GitHub ya Export ka option nahi dikh raha, toh niche diye gaye steps ko dhyan se padhein:

## 🚀 GitHub Export Kaise Dhundein:
1. **Toolbar check karein**: Editor ke sabse upar, daayein (top-right) kone mein jahan **"Publish"** button hai, uske theek left mein icons dekhiye.
2. **"Cat" (Billi) Logo**: Agar screen chhoti hai toh wo icon chhip gaya hoga. Browser mein **"Desktop Site"** on karein.
3. **Menu check karein**: Agar Desktop Site par bhi nahi hai, toh **Avatar (Aapka profile icon)** par click karein ya toolbar mein **"..." (Teen dots)** dhundein. Wahan "Export to GitHub" ka option milega.
4. **Connect**: GitHub account link karein aur **"Push"** dabayein.

## 🚀 Free Hosting (Bina Laptop ke):
1. Code GitHub par jaane ke baad, apne mobile browser mein `vercel.com` par jayein.
2. "Continue with GitHub" karke apna project select karein.
3. **Deploy** dabayein. 2 minute mein aapko ek public link (`.vercel.app`) mil jayega jo kisi bhi mobile par chalega.

## 📲 Install on Mobile:
- **iPhone**: Safari mein link kholein -> **Share (📤)** -> **"Add to Home Screen"**.
- **Android**: Chrome mein link kholein -> **3 dots** -> **"Install App"**.
