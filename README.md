# Finovo - Troubleshooting Guide 📱

Agar aapko install karte waqt abhi bhi **"Laptop"** wala icon dikh raha hai, toh iska matlab hai ki aapka Chrome browser purana icon cache se dikha raha hai.

## 🚀 Laptop Icon Kaise Hatayein (Cache Clear):
1. **Chrome Settings**: App link par jayein, Chrome ke upar **3 dots (⋮)** par click karein.
2. **Settings (⚙️)** > **Site Settings** > **All Sites**.
3. Apne app ka link (cloudworkstations.dev wala) dhundein.
4. **"Clear & Reset"** ya **"Delete all data"** par click karein.
5. Ab link ko phir se refresh karein. Ab naya **3D Wallet** icon dikhega!

## 🚀 GitHub Export:
1. **Desktop Site**: Mobile Chrome mein **"Desktop Site"** mode on karein.
2. Toolbar mein **Billi (Cat)** logo par click karein.
3. **Push** dabayein.
