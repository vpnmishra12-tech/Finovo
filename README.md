
# SmartKharcha AI - User Guide 📱

SmartKharcha AI ek **Progressive Web App (PWA)** hai. Aap ise bina Play Store/App Store ke apne phone par install kar sakte hain.

## 🚀 How to Host for FREE (Bina Laptop ke):
Agar aapko dusre mobile mein ye app chalani hai, toh ye steps follow karein:

1. **GitHub Account**: Mobile browser mein `github.com` par jayein aur free account banayein. Ek naya "Repository" banayein jiska naam rakhein `SmartKharcha`.
2. **Code Upload**: Apne is project ke saare files ko GitHub repository mein upload kar dein.
3. **Vercel Deployment**: 
   - `vercel.com` par jayein aur GitHub se sign-in karein.
   - "Add New Project" par click karein aur apna `SmartKharcha` repo select karein.
   - **Deploy** button par tap karein.
   - 2 minute mein aapko ek public link mil jayega (e.g., `smartkharcha.vercel.app`).
4. **Link Share**: Is link ko kisi bhi mobile par WhatsApp ya message se bhejein.

## 📲 Install on iPhone (Safari):
1. **Safari Browser** mein apna naya link kholiye.
2. Neeche **"Share"** button (square box with arrow 📤) par tap karein.
3. List mein neeche jayein aur **"Add to Home Screen"** (+) select karein.
4. Ye aapke iPhone mein asli app ki tarah icon ban jayega.

## 📲 Install on Android (Chrome):
1. Browser mein apna link kholien.
2. Menu (3 dots) par jayein aur **"Add to Home Screen"** ya **"Install App"** select karein.
3. Ye aapke phone mein asli app ki tarah icon ban jayega.

## 📦 How to get .APK (Android):
Agar aapko asli `.apk` file chahiye:
1. Apni app ko Vercel par host karein (upar wale steps se).
2. [PWA2APK](https://www.pwa2apk.com/) site par jayein.
3. Apni app ka public URL daalein aur APK download karein.
